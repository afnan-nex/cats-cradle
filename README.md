# 🧶 Cat's Cradle — Interactive String Art

[Live Demo](https://afnan-nex.github.io/cats-cradle/index.html)

**Cat's Cradle** is an innovative, interactive web application that brings the classic string game into the digital realm. Using real-time hand tracking, it allows you to weave and manipulate virtual strings through simple hand gestures, creating mesmerizing string art directly in your browser.

## 🌐 Live Demo

Check out the live version of the project here:  
👉 [**Open Cat's Cradle**](https://afnan-nex.github.io/cats-cradle/index.html)

## ✨ Features

- **Real-Time Hand Tracking**: Tracks your hand movements via your webcam to control the virtual strings.
- **Interactive String Art**: Generate beautiful, dynamic geometric patterns as you move your hands.
- **Performance Monitoring**: Built-in FPS (Frames Per Second) counter to track rendering performance.
- **Seamless Initialization**: Clear UI prompts for camera access and tracking initialization.

## 🚀 How to Use

1. Open the [Live Demo](https://afnan-nex.github.io/cats-cradle/index.html) in a modern web browser (Chrome, Edge, or Firefox recommended).
2. Click to allow **Camera Access** when prompted by your browser.
3. Wait a moment for the hand tracking to initialize.
4. Place your hands in front of the camera and start moving to create your string art!
5. If the camera fails to start or is blocked, simply click the **Retry Camera** button.

## 🛠️ Technologies Used

- **HTML5 / CSS3**: For the structure and styling of the interactive UI.
- **JavaScript**: For the core logic, animations, and interactivity.
- **Hand Tracking API**: For real-time webcam hand gesture recognition.
